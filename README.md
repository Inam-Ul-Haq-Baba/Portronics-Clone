# 🛍️ Portronics Clone (Responsive Website)

A fully responsive clone of the Portronics website built using **Tailwind CSS** and **Vite**.
This project replicates a modern e-commerce UI with a clean design and a mobile-first approach.

---

## 🌐 Live Demo

https://inam-ul-haq-baba.github.io/Portronics-Clone-Tailwind/

---

## 🚀 Features

* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* 🎯 Mobile-first layout implementation
* 🎨 Clean and modern UI inspired by Portronics
* ⚡ Fast performance using Vite build tool
* 🧩 Reusable and well-structured components
* 💡 Smooth and user-friendly interface

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **Vite**
* **PostCSS**

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Inam-Ul-Haq-Baba/portronics-clone.git
```

2. Navigate to the project folder

```bash
cd portronics-clone
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Build for production

```bash
npm run build
```
