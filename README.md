# Portronics Clone (Responsive Website)

A fully responsive Portronics website clone built using Tailwind CSS and Vite.  
This project follows modern UI practices and a mobile-first design approach.

---

## 🚀 Tech Stack

- HTML5
- Tailwind CSS
- Vite (Build Tool)
- PostCSS

---

## 📱 Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Mobile-first layout implementation
- Clean and modern UI
- Optimized production build using Vite
- Structured project architecture
